# <a name="is-there-a-push-to-talk-button"></a>是否有 [點擊對話] 按鈕？

在您的視圖右下角有一個麥克風按鈕，可用來切換您的聲音是否與其他使用者共用。 或者，您也可以使用鍵盤快速鍵 **Ctrl + Alt + T** 來切換此設定。 
 
當您說話時，如果圖示閃爍，您的麥克風可以正常運作！
