# <a name="creating-an-event"></a>建立事件

這是在 AltspaceVR 中建立事件的逐步指南。 強烈建議您在 AltspaceVR 中參與數個事件，以瞭解其運作方式。 查看 [AltspaceVR 事件行事曆](https://account.altvr.com/events) ，以取得所有事件的清單。

在本文中，您將了解：

* AltspaceVR 事件術語
* 建立您的活動和事件空間或世界
* 使用投影片簡報的多媒體主控台
* 使用影像來為您的活動加上商標
* 使用世界編輯器/套件新增3D 物件
* 如何錄製或即時資料流我的活動

## <a name="altspacevr-event-terminology"></a>AltspaceVR 事件術語

以下是您必須熟悉才能建立事件和事件空間的條款：

</br>

| 詞彙 | 定義 |
|---|---|
| 全球大樓 | AltspaceVR 提供建立和自訂虛擬世界的能力。 AltspaceVR 主機支援檔、但是通道和全球活動，協助您 [深入瞭解如何取得協助並開始打造](../world-building/world-building-faq.md)。 |
| World | 世界是 AltspaceVR 中的虛擬空間。 它可能是辦公室或大量的山地範圍。 它是可高度自訂的環境。 它位於 Universe 內部，而該 Universe 可能有多個領域。 如果您想要有多個事件空間可用於特殊事件、訓練中心以及不同的會議空間來 spice 專案，請在相同的 Universe 下新增世界，以將它們組合成一個群組。 |
| 宇宙 | AltspaceVR world 大樓日常用語中的 universe 代表您的世界分類。 每個 universe 都可能保有多個領域。 世界會繼承 universe 的設定，讓您輕鬆地將人員新增至多個領域和其他控制功能的允許清單。 如需詳細資訊，請參閱 [管理您的世界](../world-building/managing-worlds.md) 。 |
| 範本 | 範本 (或空間範本) 是一種預先製作的世界或環境，可以使用，而不是使用世界建築功能來建立。 AltspaceVR 提供各種不同的體驗和事件範本。 |
| 事件空間 | 事件空間是 AltspaceVR 世界的同義字。 一般而言，它是指用來裝載事件的世界。 |
| 網站 | 「網站」的參考是 [AltspaceVR 的網站](https://altvr.com/)。 透過電腦或平板電腦上的 [ [事件] 網頁](https://account.altvr.com/events/my) ，而不是透過您的 VR 裝置，通常可以更輕鬆地建立和編輯事件。 您也需要存取網站以進行 [世界大樓](../world-building/managing-worlds.md) 。 |
| 內容相關角色 | 內容[角色](../getting-started/roles.md)是由事件建立者或 world builder 所指派。 這些角色為使用者提供世界或事件空間的額外特色和功能。 目前，這些包含主機、展示者、仲裁者、試驗 (航班) 、Terraformer (世界建立) 和僅限擴音器。 這些可個別或全域指派，讓每個人都能在事件空間或世界擁有相同的角色。 |
| 消費者介面 (UI) /Menu | 當您在 AltspaceVR 的世界中，您的螢幕左側和右邊都有功能表。 具有 AltspaceVR 標誌的圓形或主功能表會開啟主要使用者介面 (UI) 或功能表，以存取不同的畫面來探索 AltspaceVR 和自訂您的體驗。 您可以在正確的畫面大小找到選擇性的 UI 專案，而且通常會包含「世界編輯器」和「主機工具」。 這些會開啟，並與您的資料指標按一下以進行互動。 |
| SDK/深入 | 這些是與軟體發展工具組和 [混合現實延伸](../world-building/using-mixed-reality-extensions.md) 模組相關聯的世界大樓詞彙，用來將功能新增到全球大樓體驗。 這些通常適用于更先進的使用者。 |

## <a name="understanding-events"></a>瞭解事件

AltspaceVR 可讓您輕鬆地建立一個世界，讓您可以使用預先製作的範本空間來針對您的特定需求使用或自訂，或從頭開始建立世界。 本文將討論如何使用預先製作的範本來建立您的活動。 下列各節說明如何使用 [影像來標記您的活動](#branding-your-event-with-images) ，並 [使用世界編輯器和套件新增3d 物件](#adding-3d-objects-using-world-editor-and-kits) ，包括進一步自訂的秘訣。 如需從頭開始建立自訂世界的詳細資訊，請聚會 AltspaceVR 中的「世界建築」教學課程，並 [查看全球大樓支援檔](../world-building/world-editor-getting-started.md)。

AltspaceVR 提供兩種方式來為您的活動建立空間。

* **一次使用：** 建立事件並選取範本世界。
* **重複使用：** 建立世界空間並將它匯入事件中。

建立事件的程式對兩者都相同，但有一個例外： [建立世界，然後將它匯入](../world-building/managing-worlds.md) 為重複使用的事件空間。 您也需要知道：

</br>

| 詞彙 | 定義 |
|---|---|
| 全球複製 | 針對重複的事件使用世界空間允許持續自訂。 正負號、影像、產生點及其他自訂都會從事件維護到事件，而不是每次自訂事件空間。 |
| 自訂事件世界 | 建立事件時，會將範本或匯入的世界複製並鎖定至該事件。 您可以對事件環境進行變更，而不會影響到原始世界，反之亦然。 請考慮新增假日裝飾、圖片或其他裝飾性的一次性使用世界編輯器元素，讓事件空間更有樂趣且適合事件。 |

除非更新事件空間，否則對原始世界的修改將不會在事件世界內：
1. 使用 [事件] 網頁上的 [ **重新匯入世界** ] 按鈕。
2. 允許同步處理完成2-3 分鐘。 
3. 如果事件空間未變更，請移至 [ **設定] > [適中] > 重設空間** ，以重設事件空間。 您會將空間重設為世界 builder！

如果您有技術問題，例如元素未正確載入，請移至 [AltspaceVR] 功能表，並選取 [ **設定] > 適中 > 重設空間** 來重設事件空間，並查看新的變更。 2D 中的 Windows 電腦使用者可以使用鍵盤快速鍵：在 AltspaceVR 中使用 **CTRL + ALT + R** 來快速重設空間。

## <a name="creating-your-event-and-event-space-or-world"></a>建立您的活動和事件空間或世界

以下是針對單次或重複事件建立事件的逐步指示。 藉由選取範本或匯入事件的世界。 

> [!NOTE]
> [AltspaceVR 事件] 網頁會透過使用綠色問號按鈕，提供表單各方面的指示。 將游標移至其上以取得特定的指示。

在 AltspaceVR 網站上的 [ [事件 > 我的事件](https://account.altvr.com/events/my) ] 頁面上，選取 [ **排程事件** ] 或直接移至 [ [在 AltspaceVR 中建立事件](https://account.altvr.com/events/new)] 網頁。

1. **活動標題：** 輸入事件的名稱。 在 AltspaceVR 的網站和全球介面中，將它保持在不同的行事曆觀點，讓它更加精確。 請嘗試將您的標題保留在23個字元（包括單字之間的空格）。
2. **描述：** 輸入事件的描述。
    * 描述中必須至少有10個字元，否則您的事件將不會建立。
    * 在段落之間加入空白空間。
    * 若要將連結新增至事件的 Facebook、但是、網站或其他資源，請使用下列格式 (此 Markdown 只適用于網站上的活動升級頁面，這不會在 AltspaceVR 功能表中正確呈現) ： `[Event Name](http://example.com/)`

3. **開始日期/結束日期：** 設定開始時間，並確定結束時間晚于開始時間。
4. **類別：** 選擇最能描述您的事件種類的類別。 
5. 將事件設定為 [ **私** 用] 或 [ **公用**]。
    * 可見度： AltspaceVR 網站事件行事曆的 [ [全部]](https://account.altvr.com/events/all) 索引標籤上會顯示公用事件，並開放給大眾。
    * 私人事件不會顯示在 AltspaceVR 事件行事曆上，需要事件 URL 才能輸入事件空間。
    * 如果您想要「新增為主要事件」，事件必須設定為「公用」。
6. **選取範本：** 網頁右側是 AltspaceVR 中可用範本的縮圖影像清單。 有遊戲室、辦公室、會議空間、簡報空間和有趣的聚會空間。 選取看起來很有趣的一個。 如果您不喜歡，請繼續使用新的範本來建立新的事件。 如果您想要建立自己的自訂事件世界，請選取 **酷炫的天空** 作為預設範本，並遵循下列指示以匯入您的世界。
7. 選取 [ **ADVANCED OPTIONS** ]

### <a name="advanced-options"></a>進階選項

1. **升階：** 每個事件都需要兩個品牌的影像 (這些影像會出現在 AltspaceVR 網站，且不會出現在 AltspaceVR) 內的事件中：
    * **磚：** 圖格影像必須為1920x1080 圖元。 此影像將會調整為各種縮圖大小，並顯示在 AltspaceVR 事件行事曆上與其他事件。 確定影像很清楚，而且沒有太多文字。 請勿使用您未擁有的受著作權影像/標誌。
    * **橫幅：** 橫幅影像必須是1920x576。 這會視需要自動調整大小，並顯示在事件資訊或網頁上。 半透明的重迭涵蓋影像的較低25%。 避免將文字放在該區域中。
    * 另一個 (更快速的) 選項是將 **磚影像複製到背景橫幅影像**，這也會使用您的磚影像作為橫幅影像。 試試看： ) 。 如果看起來沒問題，您可以建立新的橫幅影像。
2. **在 VR 中：** 本節包含在 AltspaceVR 應用程式內的事件期間適用于虛擬體驗的功能：
    * **預設內容相關角色：** 綠色問號會列出 *事件中所有觀眾成員* 可用的特定角色。 最常見的 *megaphone_only*。 新增此專案，將 [主機工具] 按鈕底下的 [增強 My Voice] 選項提供給所有觀眾成員，如此一來，如果是大空間，就可以在您的活動中聽到。 如果您的活動需要航班，請新增 *試驗*。 若要同時新增它們，格式為： *megaphone_only、試驗*。 每位使用者都必須前往 [設定/輸入/即時]，在 AltspaceVR 應用程式中啟用動態模式。
    * **指示：** 這些 [指示](../world-building/adding-welcome-messages.md) 適用于在空間抵達時產生歡迎影像的文字。 使用者必須選取 [確定] 才能將其從其視圖中移除。 這通常是用來提供物件的指示，例如「請在受邀說話之前保持靜音」或「歡迎使用 XYZ 活動，我們將在此討論 ABC」。 這並不是必要的，因此請謹慎使用。 您也可以使用 Markdown 來新增色彩和字型調整大小、更多詳細資料： [http://digitalnativestudios.com/textmeshpro/docs/rich-text](http://digitalnativestudios.com/textmeshpro/docs/rich-text)
3. **Advanced：** 以下是事件的 advanced 功能的快速說明 (其中某些功能在您建立事件和編輯事件) 之前將不會顯示：
    * 系統 **管理員：** 系統管理員是您信任的 AltspaceVR 使用者，可協助您管理您的活動。 它們可能是您的 cohosts 或備份主機。 藉由將他們的使用者名稱新增至系統管理員清單，即可：
        * 變更事件標題、描述及其他功能。
        * 為仲裁者、展示者和其他角色新增和移除內容相關角色。
        * 刪除事件。
    * **群組：** 使用下拉式功能表選擇您的私用 [群組](group-features.md) 。 只有在您的帳戶已建立/已新增至群組時，才會顯示。
    * **口號：** 這個簡潔的句子會顯示在事件網頁的標題底下。
    * **從世界匯入：** 若要使用已建立的全球事件空間來重複使用，這是用來將該世界的版面配置和自訂匯入您事件的表單區段。 請注意下列考慮：

    匯 **入您自己的世界：** 若要匯入您所建立和擁有的世界：
    * 選取下拉式箭號，以查看您所建立的世界清單。
    * 選取世界。
    * 完成事件表單的其餘部分之後，請至少等候2分鐘，再造訪 AltspaceVR 中的事件空間，以確保資料庫資訊已更新並產生世界。
    
    **使用其他人的世界：**
    * 洽詢全球擁有者，要求他們在個別世界上使用與 friend 的 **共用** 功能，與您分享。

3. **YouTube 影片識別碼：** 在事件的網頁上顯示，這會將 YouTube 事件尾端或影片新增至事件網頁，而非世界中。 您只需要 URL 的影片識別碼部分： dQw4w9WgXcQ
4. **Twitter 控制碼：** 這會將您的 Twitter 串流新增至事件的網頁。 如果 Twitter 帳戶是個人的，且與事件或關聯無關，您可能會過度共用。 您只需要控制碼： @elonmusk
5. **內容相關角色：** 您可以在此控制事件主機、展示者、仲裁者，以及事件內其他角色的超級電源或內容 [相關角色](../world-building/granting-roles.md) 。 若要加入它們，請在其 AltspaceVR 使用者名稱中輸入，然後在下拉式功能表中為他們指派角色。 若要移除它們，請選取核取方塊，以移除並儲存事件網頁。 注意：如果您想要授與展示者審核功能，您必須將他們新增為展示者和仲裁者。 目前提供下列內容相關角色：
    * 展示 **者：** 將 [Front Row 功能](../faqs/front-row-events.md)新增至 AltspaceVR 中的個別介面，以新增這些功能。
    * **仲裁者：** 將仲裁功能新增至個別的介面，包括將每位出席者的文字、全域靜音以進行事件，或將其從事件中移除的功能。 再次新增展示者，如果您希望他們擁有審核許可權，請將其授與審核角色。
    * **僅限擴音器：** 透過 [主機工具] 按鈕，將 [增強 My Voice] 按鈕加入其介面中。
    * **Terraformer：** 將 [世界編輯器] 按鈕加入其介面中。
    * **試驗：** 新增該個人的航班功能。 注意：他們必須在設定/輸入/即時啟用此功能
    * **音樂表演：** 新增與協同和音樂活動相關聯的功能和功能。
6. **封鎖列出的使用者：** 如果您想要封鎖使用者的存取權，或先前被仲裁者或展示者從事件中移除的使用者，且該事件重複，將會列出封鎖的使用者名稱清單。 事件主機或系統管理員可以隨時新增或移除封鎖列出的使用者。

完成表單並選取三次之後，請選取 [ **建立事件**]。

> [!IMPORTANT]
> 如果未正確建立您的事件，請確定您的描述中至少有10個字元，且/或您已選取範本，範本的名稱將會從白色變成藍色（若已選取）。

## <a name="event-page-actions"></a>事件頁面動作

在 [事件] 網頁上，您可以執行下列動作和選項：

1. 選取 [ **編輯** ]，對事件設定進行變更。
2. 選取 [ **結束事件** ] 以結束事件 (如果您的事件提早進行，例如) 。
3. 選取 [ **設定為草稿** ]，將您的事件設定為草稿而非使用中。
4. 如果您已對已匯入的世界進行更新，您將需要選取 [ **重新匯入世界** ]，讓這些變更出現在事件中，別忘了在事件中重設空間： ) 
5. 選取 [ **重複事件** ]，將事件複製到未來的事件。 這項複製將會出現在 [我的活動]/[我的草稿] 事件中，您必須更新新的日期/時間，然後從該處啟動它。
6. 選取 [ **新增為主要事件** ]，將您的事件設定至行事曆清單。
7. 如果您想要完全刪除事件 (您無法復原此) ，請選取 [ **刪除事件** ]。

當您準備好時，請移至 AltspaceVR 內部介面中的 [ **事件] > [我的事件** ] 功能表，然後輸入事件空間以重設空間、檢查變更或繼續自訂。

> 這是只適用于此事件的事件空間複本和自訂。 如果您花費很多時間來自訂事件的空間，則為 **重複使用事件** 建立世界可能比較好。

## <a name="branding-your-event-with-images"></a>使用影像來為您的活動加上商標

將影像新增至您的活動空間或公司的標誌與商標，甚至是您自己、您的小組或其他視覺內容的照片，都很容易 AltspaceVR。

映射需求如下：
* 接受 JPG 或 PNG 檔案
* 影像大小上限：1920x1080
* 解決方式： 72 DPI
* 檔案大小：每個檔案小於 250 KB

此程式牽涉到將影像上傳到您的 [AltspaceVR 相片](https://account.altvr.com/photos)，然後使用世界上全球編輯器中的相片來匯入，並將它們放入事件世界空間中。

1. 前往 AltspaceVR 網站上的 [相片](https://account.altvr.com/photos) 。
2. 選取 [上傳]。
3. 選取 **[流覽** ]，在您的電腦上開啟對話方塊，然後找出並選取優化的映射。
4. 選取 [開啟]  。
5. 選取 [ **建立相片**]。
6. 針對您的所有映射重複此步驟。

在 AltspaceVR 和事件世界中，將您自己放在想要放置影像的附近。  (如果您要為 **重複的使用事件** 建立此專案，請確定您在世界各地，而不是在事件空間！ ) 

1. 在使用者介面上，選取右下角的 [ **世界編輯器]/[編輯器] 面板**
2. 按一下底部的 [ **我** 的] 索引標籤。
3. 選取 [ **相片**]。
4. 相片會依上傳的時間順序儲存，因此上傳的最新影像應該是清單中的第一個影像。 選取它以將它加入至世界。
5. 使用您的裝置游標抓取影像，並將它放在您想要的位置。
    * 您可以使用控制器來調整它的大小， (通常會在 VR 中以水準方式將您的捲動方塊移至觸控板，或在2D 模式中使用您的滑鼠) 。
    * 若要微調影像的位置和旋轉，請在編輯器中選取影像上的 **齒輪** 符號，然後適當地設定旋轉和位置。
    * 您也可以使用調整功能調整映射大小。

> [!NOTE]
> 當編輯模式為 on (橙色文字) 時，您將會自動擁有飛出的能力，讓您可以更輕鬆地將影像放在事件世界的更高位置。

6. 針對每個影像放置重複此步驟。
7. 當影像設定妥當時，如果您未個別鎖定，請選取 [ **全部鎖定** ]，然後關閉編輯模式並關閉編輯器。
8. 每個事件最多隻使用五個相片。

## <a name="adding-3d-objects-using-world-editor-and-kits"></a>使用世界編輯器和套件新增3D 物件

AltspaceVR 可讓您使用世界編輯器將3D 物件匯入至世界中。 目前有許多世界編輯套件可供使用，而且有更多相關資訊。 有太空船、rockets、動畫動物、sculptures、樹狀結構、植物以及許多不同的物件。 或者，您也可以將自己的套件新增至公用集合或您自己的個人使用方式，或考慮使用其中一個 [ALTSPACEVR 深入應用程式](../world-building/using-mixed-reality-extensions.md)，將互動物件帶入您的世界，例如頭盔、swords、翅膀、服裝、頭銜或其他互動物件。 預期即將擴充的物件清單。

此處的指示著重于使用套件中已有的專案。 如需在 AltspaceVR 中新增和匯入3D 物件的詳細資訊，請參閱「[如何將 glTF 模型匯入我的世界？](../world-building/importing-models.md) 」 和「 [如何? 上傳自己的套件？](../world-building/uploading-custom-kits.md)」。

若要將現有的3D 物件從世界建築套件新增至您的事件世界，請開啟 [世界編輯器/編輯器] 面板並啟用編輯模式：

1. 探索您想要匯入之物件的套件，然後選取它。
2. 選取具有游標的套件物件，並將其定位。 在 [世界編輯器] 中使用物件的齒輪圖示，即可使用 [位置]、[旋轉] 和 [調整大小] 選項來進行手動數位輸入。
3. 當物件放置在正確的位置時，請按一下鎖定圖示來將它鎖定。
4. 如果您想要的話，請新增另一個物件。 在新增3D 物件時，請務必審慎，讓您的選擇不超過五個不同個別套件的專案，以獲得更佳的 Oculus 使用者支援。 請勿在世界各地使用超過200的物件。
5. 完成時，鎖定所有專案，關閉編輯模式，然後關閉 [世界編輯器]。 四處。 邀請幾名測試人員造訪並簽出，並準備好您的絕佳活動！
